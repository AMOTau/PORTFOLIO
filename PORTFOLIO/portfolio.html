AbilityAssist

Short description of AbilityAssist.
I am currently working on this software project with my team. We are creating a web application for visually impaired students of Tshwane University of Technology, which will help them navigate their way around the campus. What it does is that the user will interact with the application through voice recognition. The user tells the application where he/she wants to go, then the system calculates the estimated distance and arrival time to that certain place, and also guides them using voice over assistance.
It was built using Python, Django Framework, Bootstrap, HTML, CSS, JavaScript and PostgreSQL. It taught me how to use python as a programming language and how to use PostgreSQL, and I am currently advancing my knowledge on it.

It is currently not on GitHub as it is still under development.
